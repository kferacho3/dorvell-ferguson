# Dorvell Ferguson — Media Categorization Guide

_A short guide for tagging photos and videos so they land in the right place on the site.
Share this with anyone sending new media. Last updated 2026-07-08._

---

## The one rule to remember

**A single photo or video can belong to more than one category.** You are not forced to pick
just one. In the Curation Studio (the private `/studio` page) each item has:

- **One primary category** — the main bucket it lives in.
- **Any number of additional categories** — everything else it also fits.

The site uses every category an item is tagged with. So a photo tagged **Music** *and*
**Photojournalism** shows up when a visitor filters by either one. Tag generously — if it
genuinely fits two or three categories, add them all.

## The full category list

| Category | Use it for |
|---|---|
| **Portraits** | Presence, identity, expression — a person is the subject. |
| **Headshots** | Clean, professional, single-subject headshots (acting, corporate, model books). |
| **Studio** | Anything shot in a controlled studio with lighting setups / seamless / backdrop. |
| **Fashion** | Styling, wardrobe direction, lookbooks, editorial fashion. |
| **Modeling** | Dorvell's own modeling work and model-book frames. |
| **Runway** | Runway shows and fashion-week walks. |
| **Music** | Concerts, live performance, stage light, artists, crowd energy. |
| **Athletics** | Sports and game-day coverage, action, sideline, team media. |
| **Events** | Atmosphere, guests, and story from an event (not sports, not concerts). |
| **Photojournalism** | Story-first, documentary, culture, accuracy-driven coverage. |
| **Video** | Any moving-image / social clip (Reels, TikToks, recap videos, b-roll). |
| **Graphic Design** | Layouts, posters, designed graphics. |
| **Behind the Scenes** | On-set, prep, process, and crew moments. |

> New categories added in this update: **Headshots, Studio, Photojournalism, Video.**
> The rest already existed.

## Videos

1. **Videos are detected automatically.** When you upload a video file (`.mp4`, `.mov`, `.webm`, etc.),
   the studio marks it as a **Video** and tags the **Video** category for you. You don't have to do that step.
2. **Then add what the video is *about*** — this is the "dual categorization." A video almost always
   needs a second category:
   - A clip from a game → **Video** + **Athletics**
   - A creative / fashion / expression clip → **Video** + **Fashion** (or **Modeling**)
   - A concert recap → **Video** + **Music**
   - A behind-the-scenes reel → **Video** + **Behind the Scenes**
3. If a video fits several, add all of them (e.g. **Video** + **Music** + **Photojournalism** for a
   documentary-style concert piece).

**Rule of thumb for video: always Video + at least one subject category.**

## How to tag in the Studio (quick version)

1. Open the private Curation Studio page.
2. For each item: choose **Keep** or **Scrap**.
3. Pick the **primary category** (one tap).
4. Add any **additional categories** it also fits.
5. Choose where it should appear (Portfolio / Modeling / Projects) if relevant.
6. When everything looks right, **Finalize** — that publishes the kept, categorized set to the site.

Photos with no category are flagged "Needs category" so nothing slips through untagged.

## Tips for sending new media

- **Group by shoot** and note the shoot type in the folder name (e.g. `2026-06_rollingloud_music`,
  `2026-06_studio_headshots`). It makes bulk-tagging fast and accurate.
- **Videos and photos can live in the same folder** — the studio separates them automatically.
- If a set is genuinely mixed (a shoot that was both a **Studio** session and produced **Headshots**),
  that's fine — tag each item with everything that applies.
- When in doubt, add the extra category. Over-tagging is easy to trim; under-tagging hides good work.
